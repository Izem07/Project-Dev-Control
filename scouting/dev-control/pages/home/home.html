How to Connect

Quick guide for field setup

  1. 1 Start the Scout Ops Server from the Control Panel
  2. 2 Find the server machine's IP on your local network (e.g. 192.168.x.x)
  3. 3 Enter the IP and port 5000 above and click Connect
  4. 4 You'll be redirected to Server Health on success
💡 Using Bluetooth PAN? The server IP is typically 192.168.44.1 on the host device.
